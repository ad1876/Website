Facebook

Facebook är ett socialt nätverk där du kan göra allt från att skriva och dela inlägg, gå med i grupper, chatta, sälja saker, med mera. Tjänsten fyller en rad funktioner och är en av våra största sociala medier med 1.871 miljoner användare.

Facebook samlar information om dig som användare genom att se vad du tittar på, publicerar och vilka andra användare som du kommunicerar och intigrerar med. Köper du något genom Facebooks tjänster lagras information om ditt bankkort, autentiseringsinformation, fakturering och kontaktuppgifter.

Genom att använda Facebooks tjänster via en telefon eller dator, kan Facebook samla in information om dina enheter beroende på vilka rättigheter du har. Information som mjukvara, det vill säga att veta vilka andra appar som är installerade, kontaktlista, webbläsare, tidszon, kamera, album, mikrofon, telefonnummer, mobilnätverk, tjänsteleverantör, med mera. Om din spårningstjänst är påslagen kan Facebook även samla in information om vilka platser du befinner dig på och vilka platser du har besökt.

Genom insamling av information från dig som användare tillförs relevanta tjänster, lokala händelser samt målinriktad reklam. Samtidigt analyseras dessa tjänster om hur länge reklamen tittats på eller om den lokala händelsen uppmärksammades eller inte.

När du delar och kommunicerar med hjälp av Facebooks tjänster ansvarar du för innehållet. Vanligt när ett konto har skapats finns det en del inställningar som berör hur din profil visas för allmänheten, vem som kan kontakta dig och delningsmöjligheter. Detta betyder att all information som inte döljs för allmänheten kan kan ses eller nås via, API och offline-media, till exempel på TV. I med detta kan information som användaren publicerar laddas ner eller delas av en tredje person som kan ha ett annat syfte än det ursprungliga. Det vill säga att, det du gör offentligt på Facebook kan ses av andra användare eller personer, vilket innebär att man bör vara medveten om vad man skriver och lägger ut på Facebook.

Facebook lagrar din information och data så länge det är nödvändigt att tillhandahålla produkter och tjänster till dig som användare och andra användare. Det vill säga att information länkat till en användares konto kommer bli lagrad tills kontot blir raderat, eller tills data inte längre är nödvändig för att tillförse produkter och tjänster.

Som användaren har man möjlighet att radera eller deaktivera sitt konto. Väljer man att radera sitt konto försvinner alla dina inlägg, foton, kommentarer, gillningar, med mera. Avaktiverar man ett konto stängs kontot ner men kan återaktiveras vid annat tillfälle. Att radera ett konto innebär inte att all information om användaren tas bort, det vill säga att saker som andra användare har sparat eller laddat ner om dig finns fortfarande ute på internet.

Källa: Facebook. (2016). Datapolicy. Tillgänglig: https://www.facebook.com/policy.php (2017-12-04)
