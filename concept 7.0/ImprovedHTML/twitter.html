Twitter

Twitter är en av världens mest kända sociala medier där du med hjälp av videor/bilder och/eller 140 karaktärer, kan nå ut till hela världen på bara sekunder. Detta gör tjänsten perfekt för att få ut information snabbt. Twitter har idag 317 miljoner användare.

Twitter samlar på sig data så som ditt namn, användarnamn, lösenord (när ett konto skapas) och e-postadress. Du som användare ses alltid offentlig från andra användare och besökare. Som användare av Twitter kan ditt kontonamn dyka upp i sökresultat vilket visar en bild samt ditt riktiga namn på dig för allmänheten. Om du angett ditt telefonnummer i kontoinställningar kan Twitter använda din kontaktinfo för att skicka information om tjänsten, marknadsföring, hjälpa dig mot skräppost, bedrägeri eller missbruk.

Twitter samlar på sig metadata genom att samla information från dina tweets: vilken enhet, vilket datum du skapade kontot, din exakta position från GPS, information om trådlösa nätverk, eller 4G/ 3G antenn nära din mobila enhet eller din IP-adress. Tjänsten använder sig av denna information för att sedan anpassa det innehåll som du sen ser i ditt flöde. Exempelvis lokala trender, historier, annonser och följförslag.

Tjänsten lagrar data så som IP-adress, webbläsare, operativsystem, vilka länkar du har besökt, mobiloperatör, enhetsinformation och söktermer - allt för att ge en bättre användbarhets upplevelse. Även inbäddade Twitter flöden från tredjeparts hemsidor samlar data vid användandet av tjänsten.

Det är fullt möjligt för dig som användare att stänga av funktioner som personanpassade annonser och tillgång till plats. Detta gör du i dina kontoinställningar som automatiskt är påslaget efter att du upprättat ett konto. Detta innebär att redan från dagen då ditt konto skapades, skräddarsys upplevelsen på Twitter genom att samla in data från dig som användare. Twitter kan vid behov sälja information om dig som individ till företag inom reklambranschen.

Källa: Twitter. (2017). Privacy. Tillgänglig: https://twitter.com/tr/lists/privacy (2017-11-10)
